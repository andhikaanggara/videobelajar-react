# VideoBelajar - React & Tailwind Project

## 🚀 Tech Stack
- React 19.1
- TailwindCSS 4.1
- Axios (Fake API)
- React Router DOM

## 📌 Features
- Authentication (Login & Register)
- Admin CRUD Course
- Halaman Home, Semua Produk, Detail Produk
- Responsive Design
- Deployment via Vercel

## 🔧 Setup
1. Clone repo
2. `npm install`
3. `npm run dev`
